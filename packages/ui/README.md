# Wanderlust UI Package

Shadcn UI components modified and ready to use within the Wanderlust project.

## Usage

- Usage is the same as the regular Shadcn components.
- Shadcn CLI would not work properly due to the modified structure.
- You can still manually add components.
- Import them like this:

```tsx
import { Button } from '@wanderlust/ui/components/button';
```
