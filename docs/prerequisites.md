# Prerequisites

Before you can use this project, you need to have the following prerequisites installed on your system:

- Docker (https://docs.docker.com/desktop/setup/install/linux/ubuntu/)
- Bun (https://bun.sh/)

Although it is not necessary, we recommend using these tools:

- Beekeper Studio (Community Edition) (https://www.beekeeperstudio.io/get) or any other database client of your choice to connect to the PostgreSQL database.
- Redis Insight (https://redis.io/insight/) or any other Redis client of your choice to connect to the Redis server.
- Docker Desktop (https://www.docker.com/products/docker-desktop/) to manage your Docker containers and images.
